Generate Dynamic Cheque

Odoo does not have such feature this Odoo app helps user to print cheque dynamically, Now user able to configure and print cheque format according to different bank's cheque(check) format in Odoo ERP module.

Dynamic Cheque Report

User can print cheque according to his requirement.

Cheque Configuration for hight and width

User can define margin, font, font spacing, hight, width of cheque and according to that configuration the cheque will print.

Cheque Configuration for currency and its placing

User can allow currency as well as its position.
